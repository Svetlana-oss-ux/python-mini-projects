Fibonacci Sequence Generator
This is a beginner-friendly Python program that generates the first n terms of the Fibonacci sequence, where each number is the sum of the two preceding ones, starting from 0 and 1.

What is the Fibonacci Sequence?
Fibonacci numbers form a sequence such that each number is the sum of the two preceding ones, starting from 0 and 1. The first few terms of Fibonacci sequence is:

Program Description
The user is asked to input a positive integer n, and the program will output the first n terms of the Fibonacci sequence, separated by commas.
